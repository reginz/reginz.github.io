I am translating the articles, essays or blogs which has impressed me into Turkish. Nobody has same opportunities so I wanted to start this blog for enthusiastic minds who haven't had the chance to learn English.

If you are one of the great people whose articles I translate, thank you all for the amazing point of views. And if you didn't know one of your essays has been translated, it is probably because I tried to contact you but for some reasons I couldn't.

If you have any problems with this project, you can contact me via the 'Contact' button and we will work it out. But if not, and you want to support this project you can do it by clicking 'Support' button.
